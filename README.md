# Retail Sales Business Analysis

## Project Overview

This project performs end-to-end analysis on a retail sales dataset containing over 31,000 records.

The objective is to clean real-world messy data, analyze business performance, identify customer behavior patterns, and provide actionable business recommendations.

---

## Dataset

- 31,200 records
- Multiple products and categories
- Regional sales information
- Customer transactions
- Discounts and profitability data

---

## Data Cleaning

Performed:

- Missing Value Treatment
- Standardization of Region Names
- Standardization of Payment Methods
- Date Conversion
- Data Validation

---
